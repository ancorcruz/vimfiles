## Usage

    git clone git://github.com/ancorcruz/vimfiles.git ~/.vim
    cd ~/.vim
    rake install

### Updating

    cd ~/.vim
    rake update


## Customization

Add your local customizations in ~/.vimrc.local and ~/.gvimrc.local
